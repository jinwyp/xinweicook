Req

POST /api/orders/delivery/time

Group member

Name Type Desc
cookingType String "ready to cook" 或 "ready to eat"
isCityShanghai Boolean 是否在上海 true / false
isInRange4KM Boolean 是否在公司附近4公里内 true / false 
{
  "cookingType": "ready to eat",
  "isCityShanghai": true,
  "isInRange4KM": true
}
